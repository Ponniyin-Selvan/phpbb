New Changes to Archive Site (http://www.ponniyinselvan.net)

by thirumalaikv on Wed Oct 24, 2007 8:09 pm

All,

Just wanted to give an update on the archive site changes.

1. Messages will be archived every 30 minutes.

2. Whoever accessed the site would have noticed that the site
domain name is ponniyinselvan.net not ponniyinselvan.info. If you go
to ponniyinselvan.info you will be redirected to ponniyinselvan.net.
So if you have bookmarked it, please update to
http://www.ponniyinselvan.net.

3. Whoever used RSS would have noticed that not it will show the
full message instead of just the short message. The url
http://feeds.ponniyinselvan.net/PonniyinSelvan

4. Page look has changed and made compatible with most browsers.

5. Visitor map (right side bottom of the page) is available which
will show the geographical locations of visitors to the site. The site
is accessed all over the world, if you check the heat map you can see
the most visitors or from India and US and UK.

6. Message map also added which will show from which part of the
world the messages are posted to yahoo groups. You can check the map
herehttp://www.ponniyinselvan.net/pages ... e-map.html

7. Acronyms are identified and the tool tip is provided to see its
expanded form. When I joined the group and going through the posts, I
was initially scratching my head on seeing these acronyms and
collected very important ones and provided the tool tip for new
members. For eg., (RJC will be highlighted and moving the mouse over
it will show "Rajendra Cholan" - link
http://www.ponniyinselvan.net/messages/view/compare-ak-RRC-and-RJC-018441.html).
If there is any mistake or any important acronym is missed, please let
me know the message no.

8. Fancy stuff,

a. you can hear mp3 files from the browser itself if there is a
link to mp3 files (For eg., "Vaaram oru alayam" -
http://www.ponniyinselvan.net/messages/ ... 19038.html).
You need to have latest Adobe Flash installed

b. You tube and Google videos can be played in the message
itself, no need to click and go to you tube or google video site.
(http://www.ponniyinselvan.net/messages/ ... 19284.html,
http://www.ponniyinselvan.net/messages/ ... 15011.html)

More will be added as and when I get some time. Any feedback or
criticism are welcome.

Regards
Humble
