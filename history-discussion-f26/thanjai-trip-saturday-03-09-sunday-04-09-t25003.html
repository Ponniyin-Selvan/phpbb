Dear fRIENDS,

The programme IS PROPOSED to be ::

Dep Chennai :: 03/09 Saturday 0600 hrs

============================================================
SKIP UDAYARKUDI where we will be seeing the inscriptions on AK's
murder in a Recently renovated temple
===========================================================
Veeranam ( on the way to Meensuruti)

Gangai konda Chozha puram (without getting down from Vehicle)

Kudanthai (by pass) **

Lunch at Paradise Hotel - Tiruvalanchuzhi (13-00 hrs)

Departure :: 1400 hrs

Check in :: 10 mts freshening

Saraswathi Mahal 1430 - 1530

Big temple :: 1600 hrs

ASI office

1600 to 1900 Big temple

Going around Thanjai

Dinner 2100 hrs

Retiring 2200 hrs

** If there is no delay in departure and vehicle speed is normal, we
can see Panchavan madevi pallipadai / Pazhayarai Chozhan medu before
reaching Hotel Paradise for Lunch.
============================================
Dear Muruganandam, Jagan & Suganya,

Please notify Ram / Kamal - if you prefer to join us for Saturday
noon lunch / Darasuram trip. We can confirm this around 1200
Saturday, before reaching Kudanthai.

=============================================
Sunday 04/09

Departure :: 0800 hrs (Checkout)

0930 hrs :: Pulla Mangai (near Ayyampettai) on the way to Kudanthai

1100 hrs :: Break for Tea on the way to Tiruvalanchuzhi
(Coutesy : Swetha)

1115 hrs :: Tiruvalanchuzhi

1230 hrs :: Darasuram / Panchavan Madevi Pallaipadai **

1400 hrs :: Lunch at Kudanthai

1430 hrs :: Departure (via) Vikravandi
=================================================
1. PLEASE CARRY WATER BOTTLE.

2. INCLUDING LADIES, PLEASE BRING SOCKS AND WEAR, AS IT IS VERY HOT
IN THANJAI / DARASURAM TEMPLES - PAVED WITH UNCUT GRANITE STONES.
DIFFICULT TO SEE THESE TEMPLES BARE FOOTED.

3. PLEASE CARRY SYNOPSIS OF THE CLARIFICATIONS YOU NEED. WE WILL TRY
TO CHECK WITH ASI / KUDAVOIL DR.BALU

4. KEEP (DIGITAL) CAMERAS / VIDEO CAMS HANDY. FEE MAY BE PAYABLE
FOR USE IF PERMITTED.

5. WL TRY TO EMPLOY PROFESSIONAL GUIDE IN DARASURAM

=====================================================

Kalyana kalai kattudhu !

Anbudan / sps
